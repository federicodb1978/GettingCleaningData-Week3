##Variables

### Working activities

activity  = the variable containing the activity labels
x_test = dataset test
subject_test = identification of subjects test population
y_test = identification of the activities: Walking, Sitting, …

x_train = dataset train
subject_train = identification of subjects train population
y_train = identification of the activities: Walking, Sitting, …


### Outputs

data_complete = the tidy data set with meaningful variables names
mean_values = the mean of all the columns
standard_values = the standard deviation of all the columns

data_summary = the tidy data set with the average divided by Subject and Activity

data_summary.txt = the output file, representing the data_summary dataframe
